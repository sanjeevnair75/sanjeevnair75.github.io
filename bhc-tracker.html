How this tool works: The tracker fetches the "Recent Reported Orders / Judgments" table from the official Bombay High Court website (bombayhighcourt.gov.in) for each of the five benches. It then runs every entry through a keyword and party-name classifier to identify Indirect Tax matters — Customs, GST, Excise, Service Tax, Trade Remedy, FTP — and assigns a confidence level. The results appear in the table above, and can be exported to CSV for import into your register.

Confidence levels: HIGH two or more strong signals (e.g. "Commissioner of Customs" + "Bill of Entry") — MEDIUM one strong signal — LOW only generic tax/revenue keywords, needs manual review.

Note: Because browsers block direct cross-origin requests to government sites, this web tool uses a public CORS proxy (corsproxy.io) to route the request. If the fetch fails, use the Python script (bhc_daily_orders.py) instead — it has no such limitation.
